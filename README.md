# Pathway HomeBuyer Prototype

Hi-fi mobile dashboard prototype.

## Files
- `index.html` — main interactive prototype (loads at root URL)
- `print.html` — print/PDF version with all 5 screens + notes
- `shared.jsx`, `screen-today.jsx`, `screens-other.jsx`, `tweaks-panel.jsx` — components

## Deploy
Drag this folder's contents to a GitHub repo root, then connect to Vercel — no config needed. Static site, no build step.
